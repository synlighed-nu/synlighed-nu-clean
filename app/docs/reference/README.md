# Synlighed.nu

Et åbent projekt der arbejder for **fuld synlighed** i det offentlige Danmark med mottoet:  
**Hurtigere. Bedre. Billigere.** gennem agil reasoning og transparens.

---

## VIP / Reference filer

Denne mappe indeholder de vigtigste filer i projektet.  
**Hvis du ændrer design, navigation, farver eller struktur**, skal du bede Grok om at opdatere filerne i denne mappe – ellers bliver dokumentationen hurtigt forældet.

**Mappe:** `docs/reference/`

**Filer der ligger her:**
- `_Nav.tsx` ← Topmenu + mobilmenu (meget vigtig)
- `page.tsx` ← Forsiden
- `_ExplanationModal.tsx` (hvis den stadig bruges)
- Andre centrale komponenter efter behov

---

## Teknisk opbygning

- Framework: Next.js 16 (App Router)
- Styling: Tailwind CSS
- Sprog: TypeScript
- Deployment: Vercel
